Antes de prosseguir, é nessesário saber algumas coisas. No Python não usamos " ; ", e as funções/estruturas de tratamento não são definidas com chaves;

IF - ELSE (Estruras de decisão)

color = 'blue'
if color === "blue":
    print('tu já sabe que é igual ne menó')
else:
    print('tu fez alguma cagada...')

FOR

i = [1,2,3,4,5,6]
for index in i:
   print(index)
   #espera-se que retorne 1...2...3...4...5...6

#----------------------Usando Range()----------------

i = 0
for range(5):
    print(i)
    i = i+1
